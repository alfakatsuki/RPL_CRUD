Tugas Rekayasa Perangkat Lunak - Membuat CRUD Sederhana Menggunakan CodeIgniter

## Feature

- Login. Use admin:admin to login to crud system.
- Create.
- Read.
- Update.
- Delete.
- Simple clean front end with bootstrap and JQuery.
